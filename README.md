This portofolio is my final project for Bachelor Thesis “3D Object Visualization using Interactive Holographic Projection” or HOLOSEUM as its product name. This repository contains source codes and its documentation (in Bahasa), such as my thesis book, paper, presentation slides, posters, and others. 


# Unity-HOLOSEUM
HOLOSEUM – Hologram Museum
HOLOSEUM is a visualization game based C# for Human Civilization Development objects. In this game, you can interact with hologram object with your hand via Leap Motion hand sensing sensor. The floating hologram objects projected by 4-sided pyramid projector. If the gesture given by user matches the built feature, then the object displays the corresponding response. 

For further detail, you can see the product video in https://youtu.be/dyZz1oHDFhk.
